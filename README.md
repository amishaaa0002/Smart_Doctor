# Smart Doctor

Smart Doctor is an ongoing web application project that leverages machine learning, data analysis, and Flask to predict diseases such as Heart Disease, Diabetes, and Breast Cancer based on report results. It further integrates a Recommender System to provide personalized treatment plans, medicines, and recommendations.

## Features

- Disease Prediction: Utilizes machine learning models, including Linear Regression, to predict diseases based on report results.
- Recommender System: Offers tailored treatment plans and recommendations based on the diagnosed disease and patient health conditions.

## Technologies Used

- Python
- Flask (for backend development)
- Jupyter Notebook (for data analysis and model development)
- Node.js and npm (for frontend development)
- HTML (Frontend)

## Usage
- Predicts diseases like Heart Disease, Diabetes, and Breast Cancer based on report results.
- Offers personalized treatment plans, medicines, and recommendations.
- Enables remote health monitoring and predictions.
- Raises health awareness and educates users about disease risk factors.
- Supports medical professionals in diagnosing and treating patients.
